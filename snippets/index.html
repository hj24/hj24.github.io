MySQL

找出所有执行时间超过 5 分钟的线程，拼凑出 kill 语句，方便后面查杀

SELECT CONCAT('kill ', id, ';')
FROM information_schema.processlist
WHERE Command != 'Sleep'
	AND Time > 300
ORDER BY Time DESC;

Shell

截取有固定格式的长文本的第 n 列

cat example.txt|awk '{print $n}'
